Cacti A/B Testing

Introduction

I was given a Cacti shopping website and I use the theory behind A/B testing to determine which version of the website performs better.

I modified version B of the website to change the location of the Add to Cart button from the right of the image to make it inline with the image (just like Amazon does). Below I propose my hypotheses, and conduct statistical tests to appraise my hypotheses.

Hypotheses

Below I have proposed my Null and Alternate hypotheses for the categories of completion time, and return rate

Metric Time to Completion Return Rate
Null Hypotheses H0 : There will be no affect to the time to completion by making the change H0 : The difference in the return rate of version A and versin B will not be statistically significant
Alternate Hypotheses Ha : The time to completion rate will be less for version B because of the change to the layout of the Add to Cart button from the side to inline with the image Ha : Version A and B will have different return rate and this can be attributed to the change to the layout of where the Add to Cart button is placed along with the number of items in the cart, making it more user friendly and less confusing.

Takeaways and Conclusions

I have a couple of points to note with regard to the results received in conducting this experiment which I note below:

  • Something to consider is a lot of users when generating traffic for other people's websites, seemed to buffer the tabs. When the tabs load, the session ID is generated, which is why I think some of my results displayed a couple of outliers with respect to how much time they spent on the website.
  • A lot of users seemed to finish adding to the cart and then returned back to the homepage with a different version of the website, most likely because they were hard refreshing the tab which resulted in a couple of outliers too.
  • I also thought that changing the flow of the document to translate the Add to cart button and the number of items in the user's cart to inline with the image would have helped with the purposes of navigating the page in a quick manner. However, as the p-value spoke its truth, this was not the case.
  • Another reason for a p-value that we got indicating that the change did not really affect time to completion time is because users were looking to Add to Cart as quickly as possible. This experiment is not a true, reliable source to check actual user data
