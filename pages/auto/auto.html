CS1300 Development Project Writeup

Project Overview

For this project, we were told to create an interface using list and aggregator element using React.js. The users should be able to interact with different components of the interface and update their states accordingly. With these basic goals in mind, I created a website that displays a list of cars that can be sorted and filtered.

Introduction

In this project I used React.js to create an aggregator website of some sorts.

I chose to create a luxury car store website where users can filter and sort through the various cars available, and add them to their cart. It also totals the amount the user would have to pay the cars altogether.

The web app is also completely dynamic and users never have to refresh the page for anything. Here is a screenshot of the website in its final form:

Usability Principles

I kept a couple of usability features in mind when designing the web app. Below I outline some of them:

Clarity

Clarity is really important in the web app and so I wanted the user to immediately recognize what the purpose of each button and section is used for.
A visitor to the website can get to adding cars to their cart in no time.

Learnability

A user should be able to learn what the purpose behind each button or section of the website is without any instruction.
I designed the button and filters so that they are intuitive to use. The cart is also on the same page as the cars so the user knows exactly what is happening.

Accessibility

A user should be able to learn what the purpose behind each button or section of the website is without any instruction.
I designed the button and filters so that they are intuitive to use. The cart is also on the same page as the cars so the user knows exactly what is happening.

Aesthetic

Aesthetics are important to any website. This includes your color scheme, font style, and placement of text.
I made my website background #eee to give it a professional touch and color-coded my filter and sort buttons. A few of these touches along with changing my font to {Arial, Helvetica, sans-serif} made the website look much better and gave it a modern and minimal feel.
